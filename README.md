# Siddhesh Sarphale Portfolio

A simple, responsive personal portfolio website built with HTML and CSS.

## 📁 Project files

- `main.html`: Portfolio page (About, Experience, Projects, Certifications, Contact)
- `README.md`: Project description and usage instructions

## 💡 Features

- Clean modern UI design
- Responsive grid layout for all sections
- Accessible and mobile-friendly
- Contact links (phone + email) clickable

## 🚀 Run locally

1. Open `main.html` in your browser.
2. Optionally, serve with a local web server:
   - Python 3: `python -m http.server 8000`
   - Open `http://localhost:8000/main.html`

## 🛠️ Customize

- Update text in `main.html` to reflect your latest experience and projects.
- Change colors and typography in the `<style>` block.

## 📌 Notes

- No external dependencies; pure static HTML and CSS.
- Ideal to host on GitHub Pages, Netlify, or Vercel.
